# Legal

Templates and procedures the operator must adopt when running Nova.

Planned files (Phase 0):

- `TOS_TEMPLATE.md` — Terms of Service template
- `DMCA_PROCEDURE.md` — takedown receipt, processing, and repeat-
  infringer accounting
- `OPERATOR_CHECKLIST.md` — pre-launch legal checklist (designated
  agent registration, ToS publication, etc.)

These templates are starting points; operators should consult counsel
in their jurisdiction before going live.
